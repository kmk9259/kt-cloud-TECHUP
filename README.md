KT Cloud TECH-UP 과정에서 학습한 생성형 AI(Generative AI)의 핵심 원리와 대규모 언어 모델(LLM) 활용을 위한 파이썬 실습 데이터를 체계적으로 정리한 공간입니다.

단순한 데이터 전처리를 넘어, 고도화된 AI 모델링을 위한 자연어 처리(NLP) 기초부터 생성형 모델의 응용까지 단계별 학습 기록을 담고 있습니다.

---
## 📅 학습 커리큘럼 및 실습 내용

각 폴더에는 해당 날짜에 진행한 `.ipynb` (주피터 노트북) 실습 파일이 포함되어 있습니다.

| 날짜 | 주제 | 주요 내용 |
| :--- | :--- | :--- |
| **03-18** | **Web Crawling** | BeautifulSoup, Selenium을 활용한 웹 데이터 수집 기초 |
| **03-19** | **Text Analysis** | 텍스트 전처리 기법 및 데이터 분석 기초 |
| **03-19** | **Tokenization** | KoNLPy 등을 활용한 한글 형태소 분석 및 토큰화 실습 |
| **03-23** | **Data Augmentation** | 텍스트 데이터 증강 기법 및 대용량 데이터 처리 |
| **03-24** | **Embedding** | Word2Vec, FastText 등 단어 임베딩 벡터화 실습 |
| **과제** | **Assignment** | 교육 과정 중 부여된 개별 실습 과제 및 문제 풀이 |

---

## 🛠 주요 기술 스택
- **Language:** Python 3.12+
- **Environment:** VS Code, Jupyter Notebook, Colab
- **Libraries:**
  - 데이터 처리: `Pandas`, `NumPy`
  - 자연어 처리: `KoNLPy`, `NLTK`, `Gensim`
  - 웹 크롤링: `BeautifulSoup4`, `Requests`, `Selenium`
